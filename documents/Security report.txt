Protection against Man-in-the-middle attacks
----------------------------------------------------------------------------------------------------------
1. Password-reset 
For the password-reset feature, a JSON web token is being used to securely transmit a password reset link
    to the user. The token is signed to ensure the integrity of the claims contained within it. The chosen
    algorithm for the token, HMAC512,  is constructed from the SHA-512 hash function. Thus, in the event of
    an unwanted interception, an attacker will not be able to decrypt it. In addition, the token expiration
    time is set to 3600s, which leaves very little time for the attacker to do harm.
